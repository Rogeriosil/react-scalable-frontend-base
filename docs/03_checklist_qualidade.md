# Checklist de Qualidade (nível sênior)

## UI/UX
- [ ] Layout responsivo (mobile → desktop)
- [ ] Loading/Erro/Vazio nas telas com API
- [ ] Foco visível em botões/inputs (acessibilidade)
- [ ] Contraste e tipografia legíveis

## Código
- [ ] Componentes pequenos e reutilizáveis
- [ ] Tipagem (TypeScript) e interfaces
- [ ] Serviço de API centralizado
- [ ] Nomes consistentes e pastas organizadas

## Performance básica
- [ ] Assets otimizados (imagens comprimidas)
- [ ] Evitar re-render desnecessário (quando necessário)
