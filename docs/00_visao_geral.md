


# Documentação do Escopo — Projeto Front-end (Alteração Média)

Este repositório é um **starter profissional** para projetos de front-end com:
- UI responsiva e acessível
- Componentização
- Integração com APIs
- Otimização básica de performance
- Boas práticas de Git e organização de código

## Objetivo
Servir como base para implementar/alterar telas existentes, criar novas páginas e integrar com APIs,
com qualidade de entrega **nível sênior**.

## O que está incluso neste MVP
- Layout base com Header/Nav + Footer
- Rotas: Home, Dashboard, Usuários (exemplo com API)
- Estados de **loading / erro / vazio**
- Serviço de API centralizado com tratamento de erro e timeout
- Componentes reutilizáveis (Button, Card, EmptyState)
- Responsividade (mobile/desktop) e foco em UX

## Entregáveis sugeridos (para proposta)
1. Ajustes de UI/UX conforme layout/referências
2. Integração com API (listagem, filtros, detalhes, forms)
3. Responsividade completa
4. Padronização de componentes
5. Documentação e instruções de execução
6. Checklist de qualidade (ver docs/03_checklist_qualidade.md)
